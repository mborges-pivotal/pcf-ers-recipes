Getting Started with Pivotal Cloud Foundry

Before you look into this application we recommend you spend 15 minutes on the  Pivotal Cloud Foundry tutorial. It will teach you the Pivotal Cloud Foundry application deployments concepts.

Recipes

This application packages some simple recipes for getting your cloud native web application started. Think as a template to take you a step closer to realizing your idea.

You can use in different ways. You can copy the project and start implementing your own idea or you can use pieces or recipes in your own application.

Let's go

This application has 3 parts: User Interface (UI), Client and Server. The UI and client are packaged in the same project, but we have clear separation to give you the freedom to change the UI framework. The server is a separate project and can stand on its own as a backend for other applications.

Let's get started by clicking on the UI menu item above.
